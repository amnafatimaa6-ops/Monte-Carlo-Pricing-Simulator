📈 Interactive 3D Monte Carlo Stock Price Simulator

An advanced financial simulation project that visualizes Geometric Brownian Motion (GBM) in 3D and compares Monte Carlo pricing with the Black–Scholes analytical model — all wrapped in an interactive Streamlit dashboard.

🚀 Features
🎯 3D Monte Carlo Simulation
Visualize multiple stock price paths in an interactive 3D plot
Adjustable number of displayed paths
Time slicing for focused analysis

---

⚙️ Interactive Controls (Streamlit Sidebar)
Initial Stock Price (S₀)
Strike Price (K)
Risk-Free Rate (r)
Volatility (σ)
Time to Maturity (T)
Number of Steps & Simulation Paths

---


📊 Option Pricing
Monte Carlo pricing for Call & Put options
Black–Scholes analytical pricing
Real-time comparison

---

🎥 Matplotlib Animation
Animated stock price evolution
Dark-themed visualization
Live pricing display during simulation

---


🧠 Concepts Used
Geometric Brownian Motion (GBM)
Monte Carlo Simulation
Black–Scholes Model
Stochastic Processes
Financial Derivatives Pricing
Data Visualization (Plotly + Matplotlib)
